Help

Your notes are private

All your notes are stored only on your device — they are never sent to any server or cloud service. This also means your notes stay with the device and browser you use. Use Export to back up your notes to a file.

Text too large or too small?
Tap the ⚙️ gear icon to open Settings. Under Text, you can adjust the UI Font Size (controls and labels) and the Note Text Size (note content) independently.
About notes and formatting (formatting is optional)

Remember stores your notes as plain text — the same simple format used by basic text editors. You can write and read notes in plain text alone, and that works perfectly well.

Security tip: Do not store passwords in any plain text app, including this one. Use a password manager that encrypts them — Bitwarden and Proton Pass are good free options.

Optionally, you can switch to View mode (tap the pencil icon in the top toolbar) to see light formatting applied to your notes: bold, italic, strikethrough, colors, and checkboxes. In View mode, checkboxes appear as real tappable boxes you can check and uncheck — not just [ ] characters.

To add formatting, Remember uses small special characters called Markdown that you type directly into your note. For example, typing **word** makes word appear bold in View mode. You never have to use any of this — plain text notes work just as well, and the Speak button reads both kinds naturally.

Tips for getting started
The notes list

The notes list is the first screen you see when you open Remember. Each note shows its title and a short preview of its content.

Tap any note to open it. Tap the + button to create a new note.

To delete a note, open it and scroll all the way to the right in the lower toolbar — the trash icon is at the far right.

Use the Import / Export button on the notes list screen to back up all your notes to a file, or to restore them on a new device.

The note editor

When you open a note you start in Edit mode — you can tap anywhere in the text to position your cursor and start typing.

Tap the pencil icon (top right of the top toolbar) to switch to View mode, which shows your note with formatting applied — bold, strikethrough, colors, and checkboxes all appear as they should.

The Speak button reads your note aloud and is available in any mode. It works best in View mode — the text is spoken more naturally because formatting markers like bold and color have already been applied and are not read aloud.

To rename a note, tap its title at the top of the note.

The + button in the lower toolbar lets you quickly add text to the end (or beginning) of a note without tapping into the note body.

Scrolling the lower toolbar

The lower toolbar has more buttons than can fit on screen at once. You can scroll it left and right with your finger — just swipe horizontally on the toolbar itself.

Scroll arrows ( and ) appear at the edges of the toolbar when there are hidden buttons in that direction. Tap an arrow to scroll one step, or swipe the toolbar directly.

Less-used buttons like Print, Export, and Delete note are at the far right — scroll to reach them.

Optional setup steps

Everything in this section is completely optional. Remember works perfectly well without any of it — just open the link in your browser and start writing.

Installing on your Home Screen

On iPhone or iPad (Safari): Open the Remember web address in Safari, tap the Share button (the box with an arrow pointing up), then tap Add to Home Screen and tap Add.

On Android (Chrome): Open the Remember web address in Chrome, tap the three-dot menu, then tap Add to Home Screen.

Opening Remember with Siri (iPhone & iPad)

You can open Remember by voice with a one-time setup that takes about two minutes.

Step 1. Open the Shortcuts app (comes pre-installed — ask Siri "Open Shortcuts" if you can't find it).

Step 2. Tap + to create a new shortcut.

Step 3. Search for and add Open URLs (with an S).

Step 4. Paste this address into the URL field:
https://lynn-edu-apps.github.io/everyday-apps/remember/

Step 5. Name the shortcut Open Remember and tap Done.

From now on say "Hey Siri, Open Remember".

Creating a new note with Siri — asking for name and content

This shortcut asks you for both a note name and the note contents, then creates the note in Remember with the name you gave it.

Step 1. Open Shortcuts and tap +.

Step 2. Add Ask for Input.
Set the prompt to "Note name?" and input type to Text.
This result will be your note name.

Step 3. Add a second Ask for Input.
Set the prompt to "Note contents?" and input type to Text.
This result will be your note contents.

Step 4. Add URL. Type:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=new&name=
Then insert the NoteName variable from Step 2 (see below for how).
Then type &content= and insert the NoteContent variable from Step 3.

How to insert a variable depends on your iOS version. In recent versions of iOS, after you type an equals sign in the URL (e.g. name=), tap Select variable in the scrollable variable picker bar near the bottom of the screen. After you tap, you will see new items highlighted in blue that say Ask for Input appearing under the steps. Tap on the blue item underneath the step that contains the prompt asking for the value you want (the note name or the note contents).

If this is not consistent with what you see, ask Siri: "How do I insert a variable in Shortcuts".

Step 5. Add Open URL. It uses the URL from Step 4 automatically.

Step 6. Name the shortcut Add a note to Remember and tap Done.

Say "Hey Siri, Add a note to Remember". Siri will ask for the note name, then the contents, then open Remember with the new note ready to view.

Rather than building this shortcut yourself, you can add the ready-made version in one tap:

➕ Add this Shortcut to your iPhone

Tap the link above on your iPhone. iOS will ask you to add it to Shortcuts — tap Add. The shortcut is ready to use immediately.

If you want to build it yourself or understand how it works, the manual steps are described above. Be aware that Shortcuts editing on iOS can be finicky — building it from scratch takes patience.

Having Siri open and speak a note

This shortcut opens Remember, shows a large Speak Note button, and waits for you to tap it. One tap starts reading — no typing or searching needed.

Why a tap? iOS requires one tap to start audio in a browser. The button is large and easy to tap even for users with limited vision or dexterity.

Step 1. Open Shortcuts and tap +.

Step 2. Add URL. Paste:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=speak&name=Shopping+List
Replace "Shopping+List" with your note name (use + for spaces).

Step 3. Add Open URL. It uses the URL from Step 2 automatically.

Step 4. Name the shortcut Read Shopping List (or whatever you like) and tap Done.

Say "Hey Siri, Read Shopping List". Remember will open, show your note, and display a large Speak Note button. Tap it once to start reading. Tap the Stop button or anywhere on the overlay to stop.

Tip: you can create multiple shortcuts for different notes — one for your Shopping List, one for your Medication Reminders, etc.

Two things to know:

1. Due to browser restrictions, audio cannot start automatically — you must tap the large Speak Note button that appears. This is a one-tap experience by design.

2. The very first time you run the shortcut after clearing your browser history, the overlay may not appear. If this happens, simply run the shortcut a second time — it will work correctly from then on, because the page is now cached on your device.

Adding a new note with Siri (iPhone & iPad)

This lets you say "Hey Siri, Add a note to Remember" and Siri will ask what to write, then open Remember with your words already in a new note.

Step 1. Open Shortcuts and tap +.

Step 2. Add Ask for Input. Set the prompt to "What would you like to note?" and input type to Text.

Step 3. Add URL (not "Open URLs"). Paste:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=new&content=
The result of Step 2 connects automatically — you should see a dashed line between steps.

Step 4. Add Open URLs. It uses the URL from Step 3 automatically.

Step 5. Name it Add a note to Remember and tap Done.

Note: These instructions are for iOS 26 and later. On older iOS, steps may look different and you may need to manually insert a "Provided Input" variable in Step 3.

Having Google Assistant read a note aloud (Android)

This works the same way as the Siri shortcut — Google Assistant opens Remember with a large Speak button, and you tap it once to start reading.

One-time setup:

Step 1. Open the Google app → tap your profile picture → Settings → Google Assistant → Routines → tap +.

Step 2. Under "When I say...", type the phrase you want to use, e.g. Read my shopping list.

Step 3. Under the action, choose Open website and paste:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=speak&name=Shopping+List
Replace "Shopping+List" with your note name (use + for spaces).

Step 4. Save the routine. Say "Hey Google, Read my shopping list".

Remember will open in Chrome, show your note, and display a large Speak Note button. Tap it once to start reading.

Two things to know:

1. Due to browser restrictions, you must tap the Speak Note button — audio cannot start automatically.

2. The first time you run the routine after clearing your browser history, it may not show the overlay. If this happens, run it a second time — it works correctly once the page is cached.

Tip: create multiple routines for different notes — "Read my medication list", "Read my shopping list", etc.

Adding text to a note by voice with MacroDroid (Android)

Google Assistant Routines can open a note by voice, but they cannot ask you what text to add. For that, you need a free app called MacroDroid.

Simple way — open a specific note by voice (no extra app needed):

Say "Hey Google, open my Shopping List" and set up a Google Assistant Routine that opens:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=open&name=Shopping+List

Better way — dictate new text into a note:

Step 1. Install MacroDroid from the Play Store (free).

Step 2. Create a new Macro. Set the trigger to Voice Command and type a phrase like Add to Shopping List.

Step 3. Add an action: User Prompt → Input Dialog. Set the prompt to What would you like to add?

Step 4. Add another action: Device Actions → Open URL. Paste:
https://lynn-edu-apps.github.io/everyday-apps/remember/?action=quickadd&name=Shopping+List&content={input_var}
Replace Shopping+List with your note name (use + for spaces).

Step 5. Save the macro. Now say "Hey Google, Add to Shopping List" — MacroDroid will ask what you want to add, then open Remember with your text already inserted into the note.

If you have installed Remember on your Android home screen as a PWA, you can long-press the Remember icon to see quick action shortcuts:

• New Note — creates a new empty note immediately

Sharing text into Remember

Remember can receive text shared from other apps (web pages, emails, messages). In any app, tap the Share button and look for Remember in the share sheet — it will create a new note with the shared text.

The share sheet option appears after installing Remember on your home screen.

URL actions (for Siri Shortcuts)

Create a new note:
…/remember/?action=new
…/remember/?action=new&content=Your+text+here
…/remember/?action=new&name=Note+Title&content=Your+text+here
When both name and content are provided, both are required (neither can be blank) and the note name is permanently set — it will not change when you edit the content.

Open a note:
…/remember/?action=open&name=Shopping+List

Open a note and show the Speak button:
…/remember/?action=speak&name=Shopping+List
Opens the note and shows a large full-screen Speak Note button. One tap starts reading aloud. Ideal for use with Siri Shortcuts — see the setup steps in the Optional Setup section above.

Quick-add text to a note:
…/remember/?action=quickadd&content=Your+text+here&name=Shopping+List
…/remember/?action=quickadd&content=Your+text+here&name=Shopping+List&position=2
…/remember/?action=quickadd&content=Your+text+here&name=My+Note&position=after-rule
Appends (or prepends, per your Settings) the content to the named note, then opens it. If the name matches multiple notes, the most recently edited one is used. If no match is found, a new note is created with that name.
